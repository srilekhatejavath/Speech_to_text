# Gemini Audio Transcription

A simple AI-powered application that converts recorded speech into text using the Google Gemini API and Gradio.

## Features
- Record audio from microphone
- AI-powered speech-to-text
- Accurate transcription with punctuation
- Easy-to-use Gradio interface

## Technologies
- Python
- Gradio
- Google Gemini API

## Run

```bash
pip install -r requirements.txt
python app.py
```
